# Event Management System

A Django-based web application for managing events efficiently. This system allows event organizers to create and manage events, while users can view events and register for them.

## Features
- User authentication (login and registration)
- Event creation, update, and deletion
- Event registration for users
- Admin panel for managing users and events
- Secure and scalable application

## Tech Stack
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite

## Architecture
- Follows Django’s MVT (Model–View–Template) architecture

## How to Run
python manage.py runserver
